# kai-file 文件操作模块
提供通用的文件操作功能

## 功能
- reverse_line_reader.rs 提供基于tokio的异步功能，按行倒序读取文本文件内容
  - 每行为正序，行与行之间为倒序
  - O(1)时间复杂度、O(1)空间复杂度

## 实现约定

### 术语
- 行：以换行符(\n)分隔的文本单元，支持Windows风格的\r\n
- 缓冲区：用于从文件末尾向前读取数据的临时存储区

### 倒序行读取机制
- 从文件末尾开始，以固定大小的缓冲区向前读取
- 在缓冲区中查找换行符来定位行边界
- 按倒序返回行内容，每行内部保持正序
- 支持流式读取，不需要将整个文件加载到内存
